projects

 startpage 
My startpage with links to my favorite websites, date and time, the weather forecast and an image that's nice to look at. The searchbar is multifunctional (one might call it 'smart') and just behaves as one'd expect it to (i.e. accepts search queries, DuckDuckGo Bangs and URLs with and without the 'http[s]://' scheme). Also supports playing certain radio channels. The images served are all webp images to improve load time.
 mailserver 
A set of config files (with secure defaults AFAIK) for a mailserver running Postfix, Dovecot and Rspamd. Also contains a whole write-up of how to set it all up (including DNS, etc). I mainly wrote all this for myself to remember all the things involved, if I need to do it again. Currently I'm also running my own mailserver.
 vplan 
A substitution plan scraper for a tool widely used by schools in Germany. The 'official' way to use said tool was annoying and a popular (3rd party) app that popped up a little later was ehh. Thus I wrote my own tool. The 'backend' is basically a bash script that does some heavy regex matching and edits some html files served by nginx, while the 'frontend' is mainly javascript logic like sorting by date, hiding old items and a filtering engine that supports regexes.
 mradio 
A PWA-esque webpage for listening to different online radio stations/streams. UI is generated on-the-fly from a config file that contains info like the stream URL or thumbnails for each channel. It's called 'mradio' ('mobile radio') because it was originally intended for mobile phones (the available apps weren't to my liking).
 huec 
A program to control your Philips Hue lights, written in C. Configuration is done in a minimalistic way, using just plain ol' header files. Apart from changing power-on state, brightness, hue, saturation (etc.), registering with the Hue Bridge API, getting info about a light from that API as well as setting predefined profiles (aka scenes) is also supported.
 remotelockd 
A 'daemon' that listens on a local port and suspends the machine when it receives a valid, one-time token. Use case: when you go to the printer and forgot to lock your device.
 shelltwitch 
A CLI tool to check which streamers are online, what they're playing (etc) using the Twitch API. Sends desktop notifications when someone's gone live using notify-send and cron. Followed streamers are also pulled from the API.
 xpowerpredict 
A Python script that tries to predict possible point loss/gain for an online multiplayer game called Splatoon 2. Spawns a window using Tkinter to display the calculated values, which sometimes aren't that accurate because the current formula tends to inflate results in special cases.
 iserv-aufgaben-rss 
A bash script that scrapes the exercises teachers give students from the school's I-Serv platform and generates an RSS feed from that.
